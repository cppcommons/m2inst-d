Jx9 and the HTTP Protocol.

Jx9 includes a powerful HTTP request parser with the ability to extract MIME headers, decode GET/POST raw queries and populate the appropriate predefined JSON objects such as $_GET, $_POST and so forth. This feature is popular for network oriented applications such as web servers or network games that embed Jx9 and need to pass HTTP informations to their Jx9 scripts. Refer to the JX9_VM_CONFIG_HTTP_REQUEST configuration option for additional information.
